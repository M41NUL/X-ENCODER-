## рҹ“Ң What is X-ENCODER?

**X-ENCODER** is a Termux-based Python source code protection tool. It encodes your `.py` files using **3 layers of obfuscation** вҖ” making them extremely difficult to read or reverse-engineer, while still being fully executable.

> Protect your scripts before sharing them. Keep your logic private.

---

## рҹ”җ Encoding Layers

```
Original .py  в”Җв”Җв–ә  Marshal (bytecode)  в”Җв”Җв–ә  Zlib (compress)  в”Җв”Җв–ә  Base64 (encode)  в”Җв”Җв–ә  Encoded .py
```

| Layer | Method | Purpose |
|-------|--------|---------|
| 1st | `marshal` | Compiles source to Python bytecode (code object) |
| 2nd | `zlib` | Compresses marshalled bytes (level 9) |
| 3rd | `base64` | Encodes compressed bytes to ASCII-safe string |

The output file is **self-executing** вҖ” no decoder needed. Just run it normally with `python file.py`.

## вҡЎ Installation & Usage

### Step 1 вҖ” Clone the repo

```bash
git clone https://github.com/M41NUL/X-ENCODER-.git
cd X-ENCODER-
```

### Step 2 вҖ” Run installer

```bash
bash installer.sh
```

The installer will:
- Update Termux packages
- Install Python, pip, git
- Request Android `/sdcard` storage permission via `termux-setup-storage`
- Show animated progress bars for each step
- Auto-launch `main.py` after a **3-second countdown**

### Step 3 вҖ” Run manually (after first install)

```bash
cd X-ENCODER-
python main.py
```
